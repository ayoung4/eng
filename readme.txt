====================================================================================
P5 + TS + Browserify Boilerplate
====================================================================================
DESCRIPTION:

Boilerplate code for a p5 sketch using typescript and browserify. 


====================================================================================
SETUP:

> npm install

install all needed npm packages

> npm start

compile and watch ./src directory, start serving index.html on localhost:8080 


====================================================================================
COMMANDS:

> npm run serve 

Starts a basic http server to serve index.html on localhost:8080.

> npm run bundle 

Compiles + bundles ./src/main.ts and all dependencies into ./dist/bundle.js. 
Watches ./src directory for changes and recompiles.
